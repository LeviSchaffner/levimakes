Raspberry PI 3D Printing

Project MIRS

Levi Schaffner · January 2025

A wrist mounted object recognition system from the game Rainbow Six Siege

This was my first project to use a Raspberry Pi and custom 3D printed parts. I modeled the enclosure in Tinkercad and 3D printed it in PLA. I used Tensorflow Lite with the Google Coco library. The device utilized a Raspberry Pi camera and a small LCD touchscreen.
